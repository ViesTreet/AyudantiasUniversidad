🏛️ Acceso al Templo del Linaje

Solo los que entienden el origen pueden pasar. ¿Cuál es el secreto?

📜 Soluciones: El Linaje de los Guardianes

Prueba 1 y 2: Definición con super()

class Guerrero(Guardian):
    def __init__(self, nombre, nivel, fuerza):
        super().__init__(nombre, nivel) # Llama al padre
        self.fuerza = fuerza

class Mago(Guardian):
    def __init__(self, nombre, nivel, mana):
        super().__init__(nombre, nivel)
        self.mana = mana

Prueba 3 y 4: Overriding (Sobrescritura)

# En Guerrero:
def accion_principal(self):
    return "El guerrero ataca con su espada!"

# En Mago:
def presentarse(self):
    return f"Soy {self.nombre}, un mago de nivel {self.nivel} con {self.mana} de mana."

Prueba 5: Polimorfismo en Acción

ejercito = [Guerrero("Ragnar", 10, 80), Mago("Gandalf", 20, 100)]

for unidad in ejercito:
    print(unidad.accion_principal())
# Salida:
# "El guerrero ataca con su espada!"
# "El guardian se mantiene en guardia." (Si el mago no lo redefinió)
